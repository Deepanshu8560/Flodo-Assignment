# Task Management Application

A full-stack Task Management Application built with Flutter (frontend) and FastAPI (backend).

## Tech Stack

- **Frontend**: Flutter (Clean Architecture, Riverpod)
- **Backend**: Python (FastAPI, SQLAlchemy)
- **Database**: SQLite (async via aiosqlite)

## Project Structure

```
├── backend/          # FastAPI REST API
│   ├── app/

│   │   ├── services/

│   └── tests/
├── frontend/         # Flutter application
│   └── lib/
│       ├── core/
│       └── features/tasks/
│           ├── data/
│           ├── domain/
│           └── presentation/
└── docs/             # Documentation
```
